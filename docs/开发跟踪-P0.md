## ✅ P0-1：活动管理页面

**目标**：在浏览器中直接创建/编辑/删除秒杀活动，无需调 API

### 后端

| 接口 | 状态 |
|------|------|
| `POST /activity/create` | ✅ 已有 |
| `PUT /activity/update` | ✅ 已有 |
| `DELETE /activity/{id}` | ✅ 新增（逻辑删除） |
| `PUT /activity/status/{id}?status=` | ✅ 新增（上架/下架，上架时自动预热库存） |
| `ActivityVO.dbStatus` | ✅ 新增字段（保留数据库原始状态，供管理后台判断按钮） |

### 前端

| 文件 | 说明 |
|------|------|
| `src/views/admin/ActivityForm.vue` | ✅ 创建/编辑活动表单页面（含图片上传） |
| `src/views/admin/ActivityManage.vue` | ✅ 活动管理列表（上架/下架/删除/编辑，用 dbStatus 控制） |
| `src/views/Layout.vue` | ✅ 导航栏新增「⚙️ 活动管理」入口 |
| `src/router/index.js` | ✅ 新增 3 个管理路由 |
| `src/api/index.js` | ✅ 新增 updateActivity/deleteActivity/updateActivityStatus 方法 |

### 验收标准
- [x] 能打开表单页面，填写活动信息
- [x] 能创建新的秒杀活动
- [x] 能编辑已有活动
- [x] 能删除活动
- [x] 能上架/下架活动（上架自动预热库存）
