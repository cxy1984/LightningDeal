## 二、详细任务说明

### 订单超时自动取消

- **业务逻辑**：用户下单 30 分钟未支付 → 自动取消订单 → 恢复库存
- **技术方案**：RabbitMQ 延迟队列（`x-delayed-message` 插件）
- **实现链路**：
  ```
  下单成功
    ↓
  SeckillOrderConsumer.handleSeckillOrder()
    → 创建订单（status=0）
    → 发送延迟消息（30min）
    ↓ 30分钟后
  handleOrderTimeout(orderId)
    → orderService.cancelOrder(orderId)
    → Redis库存 +1
    → MySQL sold_stock -1
    → 用户购买计数 -1
    → 用户标记移除
    → 用户可重新抢购
  ```
- **修复内容**：
  - `cancelOrder()` 补充了 Redis 库存恢复和用户标记移除
  - `refundOrder()` 同样补充了 Redis 库存恢复
