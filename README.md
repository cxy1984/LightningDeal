# ⚡ LightningDeal 秒杀系统

一个基于 Spring Boot + Vue3 的高并发秒杀系统，涵盖 Redis 分布式锁、MQ 异步削峰、ES 全文搜索、MinIO 对象存储等主流中间件实战。

---

## 技术栈

| 层级 | 技术 | 版本 | 用途 |
|:----:|:----|:----:|:----|
| **后端** | Spring Boot | 2.7.x | 基础框架 |
| | MyBatis-Plus | 3.5.x | ORM |
| | MySQL | 5.7+ | 持久化 |
| **缓存** | Redis | 3.2+ | 库存预热、原子扣减、分布式锁、限流 |
| **队列** | RabbitMQ | 3.x | 异步下单、流量削峰、延迟重试 |
| **搜索** | Elasticsearch | 7.x | 活动商品全文搜索 + ik 分词高亮 |
| **存储** | MinIO | 最新 | 商品图片、文件存储（可选扩展） |
| **实时** | WebSocket | 原生 | 抢购结果推送、实时成交记录 |
| **前端** | Vue 3 | 3.x | 前端框架 |
| | Element Plus | 最新 | UI 组件库 |
| | ECharts | 5.x | 数据可视化大屏 |
| | Vite | 5.x | 构建工具 |

## 本地开发环境准备

### 需要本地安装的服务

| 服务 | 版本 | 用途 | 确认方式 |
|:----|:----:|:----|:--------|
| MySQL | 5.7+ | 数据库 | `mysql --version` |
| Redis | 3.2+ | 缓存 | `redis-cli ping` |
| RabbitMQ | 3.x | 消息队列 | 访问 http://localhost:15672 |
| Elasticsearch | 7.x | 搜索引擎 | 访问 http://localhost:9200 |
| MinIO | 最新 | 对象存储（可选） | 访问 http://localhost:9000 |
| Node.js | 16+ | 前端构建 | `node -v` |
| Maven | 3.6+ | 后端构建 | `mvn -v` |
| JDK | 1.8+ | 编译运行 | `java -version` |

### 启动顺序

```
1. 启动 MySQL / Redis / RabbitMQ / ES（本地已装则跳过）
2. git clone https://github.com/你的用户名/LightningDeal
3. ld-backend/
   └─ mvn clean package -DskipTests
   └─ java -jar target/lightning-deal.jar
4. ld-frontend/
   └─ npm install
   └─ npm run dev
5. 浏览器访问 http://localhost:5173
```

## 面试价值

| 技术点 | 面试能聊什么 |
|:-------|:------------|
| Redis | 库存预热、DECR 原子扣减、SET NX 分布式锁、滑动窗口限流、库存与数据库一致性 |
| RabbitMQ | 异步削峰、消息可靠性（confirm + ack）、死信队列重试、幂等性处理 |
| Elasticsearch | ik 分词、高亮查询、索引设计、与 MySQL 数据同步 |
| MinIO | 对象存储选型对比、S3 兼容 API、预签名 URL |
| WebSocket | 实时推送方案对比、连接管理、心跳检测 |
| 高并发 | JMeter 压测、QPS 指标、瓶颈分析、优化策略 |
